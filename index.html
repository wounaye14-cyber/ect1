      ☁️ Synchronisation cloud (ordi ↔ téléphone)
      Chargement...

      Pour synchroniser tes données entre tous tes appareils, connecte une base de données Supabase gratuite. Suis ces étapes une seule fois :

      Étape 1 — Crée ton projet Supabase
      1. Va sur supabase.com et crée un compte gratuit
      2. Crée un nouveau projet (nom au choix, ex: "ect1")
      3. Dans ton projet, va dans SQL Editor
      4. Colle et exécute ce code SQL :
      CREATE TABLE ect_data (
  user_id TEXT NOT NULL,
  key     TEXT NOT NULL,
  value   TEXT,
  updated_at TIMESTAMPTZ DEFAULT now(),
  PRIMARY KEY (user_id, key)
);
      Étape 2 — Récupère tes clés
      1. Dans Supabase, va dans Settings → API
      2. Copie l'URL du projet (ex: https://xxxxx.supabase.co)
      3. Copie la clé anon public
      Étape 3 — Connecte ici

      Les données se sauvegardent automatiquement dans le cloud à chaque modification.
      Sur un nouvel appareil, connecte-toi avec le même URL et la même clé, puis clique sur Télécharger depuis le cloud.

      Étape 1 — Crée la table Supabase

      Dans ton projet Supabase → SQL Editor → New query → colle ce code → Run :

      CREATE TABLE IF NOT EXISTS ect_data (
        user_id TEXT NOT NULL,
        key TEXT NOT NULL,
        value TEXT,
        updated_at TIMESTAMPTZ DEFAULT now(),
        PRIMARY KEY (user_id, key)
      );
      👆 Clique pour copier
      Étape 2 — Récupère tes clés

      Dans Supabase → Settings → API
      Copie : Project URL et anon public key

      Étape 3 — Connecte ici
      📱 Comment synchroniser ordi ↔ téléphone
      1
      Configure Supabase une seule fois sur ton ordi (étapes ci-dessus)
      2
      Clique ⬆️ Envoyer vers le cloud pour sauvegarder toutes tes données
      3
      Sur ton téléphone, ouvre le site Netlify → ☁️ Synchronisation → entre les mêmes identifiants → ⬇️ Télécharger du cloud
      ✓
      Ensuite chaque modification se sauvegarde automatiquement dans le cloud en temps réel
